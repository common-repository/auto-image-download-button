== Description ==

### Image Download Button ###

The plugin adds a **customizable download button** automatically below every image on post/page.

You might have shared different images on your website and **you want your visitors to download them** easily without any issue.
Then this **image download button** plugin will help you to do the same, as after you install and activate this plugin on your WordPress it will **automatically add a download button** below every image on every post, page and different post types.

== Installation ==

1. Download the plugin from WordPress plugin directory (this page).
2. Upload the zip folder 'Image-Download-Button' to the '/wp-content/plugins/' directory.
3. Activate 'Auto Image Download Button' through the 'Plugins' menu in WordPress, That's it.
4. The download button will be added to every image on your blog post automatically, and you can customize it according to your needs.
